# Choosey - 先進的選課管理系統


想像一個不需複雜操作、可提前規劃、全自動但又可微調的選課系統，<br />
聽起來很夢幻嗎?<br />
但其實就在你的眼前!

專為大同大學系統量身訂做，<br />
基於Node.js的Electron應用程式，<br />
Choosey開發團隊致力於打造完美的選課體驗!

## Features
目前版本包含以下功能:

### 選課前規劃
* 查詢各班課程
* 查詢通識課程
* 一鍵切換課程列表與時間表
* 點擊課程名稱直接顯示課程大綱
* 選課購物車

* NTP網路校時
